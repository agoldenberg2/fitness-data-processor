# Fitness Data Processor

This project is a Node.js  that processes fitness data from files. It reads workout data from a CSV file and health metrics from a JSON file, then shows a summary. The project also includes has tests using Jest to ensure the file readers work correctly.

---

## Features

- Reads workout data from a CSV file
- Reads health metrics from a JSON file
- Calculates totals and counts from the data
- Includes error handling for missing or invalid files
- Uses Jest for automated testing
